# CoGamDa Study 📘

Microsoft AI Engineer 과정을 병행하며  
코딩 감각을 기르기 위한 스터디용 GitHub 레포지토리입니다.

문제는 **Microsoft Teams**에 공유되고,  
풀이 코드는 이 레포지토리에 기록합니다.

---

## 📌 Git 사용 규칙

### 1️⃣ 최초 1회만 clone
레포 참여 시 **처음 한 번만 clone** 합니다.

    git clone https://github.com/lifeiscabaret/CoGamDa-st.git

이후부터는 **clone을 다시 하지 않습니다.**

---

### 2️⃣ 문제 풀기 전에는 항상 pull
다른 팀원의 변경사항을 먼저 받아오기 위해  
**작업 시작 전 항상 `git pull`을 먼저 실행합니다.**

    git pull

---

## 📁 폴더 / 파일 구조 규칙

### 3️⃣ 기본 구조

    COGAMDA_ST/
     ┗ 본인이름/
       ┗ day0/
         ┗ problem1.py

---

### 4️⃣ 폴더 규칙
- `본인이름/` : 본인 이름 폴더는 **최초 1회만 생성**
- `day숫자/` : 문제를 푼 날짜 기준 (day0, day1, day2 …)

---

### 5️⃣ 파일 규칙
- `problem숫자.py` : 해당 day의 문제 번호 (problem1.py, problem2.py …)

---

## ✍️ 커밋 메시지 규칙

### 6️⃣ 커밋 메시지 형식

    day00 풀이1~ - 본인이름

---

### 7️⃣ 커밋 메시지 예시
- day00 풀이1 - jihyun  
- day01 풀이2 - minsu  
- day02 풀이1~3 - younghee  

⚠️ 이름은 **성 제외**, 폴더에 작성한 이름과 **동일하게** 작성해주세요.
